# Guía de Referencia del Tema por Defecto

## El Mejor Ejemplo es el Código

Las guías de desarrollo (`theme-development-guide.md` y `theme-visual-guide.md`) proporcionan la base teórica y conceptual para crear temas en la plataforma Fasttify.

Sin embargo, a menudo el mejor recurso para resolver dudas y entender cómo se aplican los conceptos en la práctica es examinar una implementación real y funcional.

### Explora el Tema por Defecto

Para una comprensión más profunda y para ver ejemplos concretos de todo lo explicado en la documentación, te recomendamos encarecidamente que explores el código del **tema por defecto**, ubicado en el directorio `/template` de este repositorio.

➡️ **Directorio del Tema Base: [`/template`](../../template)**

Dentro de esta carpeta, encontrarás:

- Una implementación completa de `theme.liquid`.
- Ejemplos funcionales de plantillas `.json`.
- Una gran variedad de secciones y snippets que puedes usar como inspiración o base para tus propios componentes.
- La estructura de `settings_schema.json` con configuraciones globales.
- Archivos de `assets` y `locales` completamente funcionales.

Considera el directorio `/template` como una "documentación viva" que siempre estará actualizada con las mejores prácticas y las funcionalidades más recientes de la plataforma.
